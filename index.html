說明


在《Apex 英雄》中和你的角色一起爭霸,這是一款免費遊玩的英雄射擊遊戲,其中擁有強大技能的傳說級角色將在戰場上一起組隊合作來在邊境上取得名聲與財富。


在這個超越大逃殺體驗的殘破而紛亂的世界中,掌握不斷增加的多種英雄、精深的戰術小隊遊戲和大膽的創新。歡迎來到新一代的英雄射擊遊戲。


重點特色：

  • 各種傳奇角色 - 掌握不斷增加的各種強大英雄,他們各自擁有不同的個性、專長與技能且易學難精。

  • 建立你的隊伍 - 選擇你的英雄並和其他玩家一起配合使用他們獨特的技能來組成終極的隊伍。

  • 戰術小隊遊戲 - 無論你是在巨大的飄浮城市中展開大逃殺,還是在競技場中進行近戰決鬥,你都得懂得臨機應變。掌握英雄的獨特技能並和隊友一起合作來找出新的戰術和強大的組合。

  • 創新的戰鬥 - 掌握各式各樣不斷增加的強大武器與裝備。你得快速行動並瞭解每種武器的節奏才能善用所有武器。另外,在限時模式中體驗不同的內容,並準備好在每個新賽季中體驗大量新內容。

  • 不斷擴展的宇宙 -《Apex 英雄》位於一個具沈浸感的世界中,這段故事將不斷演變、地圖將在每個賽季中改變且新英雄將不斷加入戰鬥。在 Apex 競賽中使用各式各樣的獨特裝扮來打響名聲並加入冒險！

此遊戲包含可選的遊戲內虛擬貨幣購買,可用於獲取遊戲內虛擬物品,包括隨機選擇的虛擬遊戲內物品。

系統需求


最低系統需求

作業系統：Windows 7 64 位元版

處理器 (AMD)：AMD FX 4350 或同等級產品

處理器 (Intel)：Intel Core i3 6300 或同等級產品

記憶體：6GB - DDR3 @1333 記憶體

顯示卡 (AMD)：AMD Radeon™ HD 7730

顯示卡 (NVIDIA)：NVIDIA GeForce® GT 640

DirectX：11 版相容顯示卡或同等級產品

線上連線需求：512 KBPS 或更快的網際網路連線

硬碟空間：22GB


建議系統需求

作業系統：Windows 7 64 位元版

處理器 (AMD)：Ryzen 5 CPU 或同等級產品

處理器 (Intel)：Intel Core i5 3570K 或同等級產品

記憶體：8GB - DDR3 @1333 記憶體

顯示卡 (AMD)：AMD Radeon™ R9 290

顯示卡 (NVIDIA)：NVIDIA GeForce® GT 970

DirectX：11 版相容顯示卡或同等級產品

線上連線需求：寬頻連線

硬碟空間：22GB
